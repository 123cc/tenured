# Tenured

完全真正开源IM项目。



## 特性

- [ ] 支持点对点消息(文字、语言、图片、小视频)，群组消息，聊天室消息

- [ ] 支持客服系统

- [ ] 支持Android/Iphone(OC/swift)/Web-WS/小程序

- [ ] 服务端全功能API

- [ ] 支持集群动态扩展

- [ ] 公有云对接

- [ ] 私有部署、私有化Docker部署。

- [ ] 支持阿里云自动化部署。

- [ ] 支持SSL/TLS加密连接。
